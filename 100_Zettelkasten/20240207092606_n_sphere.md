# $n$-sphere

The $n$-sphere, $S^{n-1}$, is the set of points $(x_1, \dots, x_n) \in \mathbb{R}^{n}$ satisfying the equation $x_1^2 + \dots + x_n^2 = 1$.

$$S^{n-1} = \{x = (x_1, \dots, x_n) \in \mathbb{R}^n / x_1^2 + \dots + x_n^2 = 1\}.$$

The $1$-sphere is a circle.
